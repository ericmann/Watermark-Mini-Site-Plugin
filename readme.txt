== Description ==

Allow the creation of multiple mini-sites within a WordPress network.

== Installation ==

Activate in the "Plugins" admin panel using the "Activate" link.

== Frequently Asked Questions ==

= Is this plugin compatible with other multi-network plugins? =

No!

You can only have one system that leverages the multiple network feature of WordPress installed.
